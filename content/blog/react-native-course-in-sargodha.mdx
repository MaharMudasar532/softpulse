## Curriculum Overview

Our [React Native course](/react-native-course-sargodha) and broader [mobile app development course](/mobile-app-development-course-sargodha) follow a structured path from zero to published apps.

### Phase 1 — Web Foundations

Before mobile, you need solid JavaScript. We cover HTML, CSS, and modern JavaScript (ES6+): variables, functions, arrays, objects, async/await, and DOM basics. This foundation prevents the confusion many self-taught developers carry for years.

### Phase 2 — React Core

React Native is built on React. You learn components, props, state, hooks, conditional rendering, lists, and forms. You build small web projects first so mobile concepts click faster later.

### Phase 3 — React Native Fundamentals

Here mobile development begins. You work with View, Text, ScrollView, FlatList, StyleSheet, and Flexbox layouts on real device screens. Navigation with React Navigation connects screens into complete app flows.

### Phase 4 — Device Features and APIs

Modern apps need more than static screens. You integrate cameras, geolocation, push notifications, and REST APIs. Firebase handles authentication, real-time database, and cloud storage — the same stack many startups use.

### Phase 5 — Publishing and Portfolio

The final phase covers production builds, app icons, screenshots, privacy policies, Google Play submission, and App Store guidelines. Students leave with deployable portfolio projects, not half-finished mockups.
